# Zielbild Datenqualitäts-Monitor

Das Bild `zielbild.png` dient als visuelle Orientierung, nicht als pixelgenaue Vorlage.

Wichtige Zielstruktur:
1. Bestehendes SaTourN-Statistik-Dashboard erhalten.
2. Filterbereich oben beibehalten.
3. Bestehende KPI-Zeile beibehalten.
4. Qualitäts-KPIs ergänzen.
5. Häufigste Pflegebedarfe als klickbare Liste anzeigen.
6. Klick auf ein Qualitätsproblem öffnet eine gefilterte Fehlerliste.
7. Fehlerliste muss die aktiven Filter für Gebiet, Ort, Typ und Kategorie berücksichtigen.
8. Fehlerliste muss per CSV exportierbar sein.
9. Kriterien-Matrix und Datentyp-Übersicht ergänzen.
10. KI-Chatbot über n8n-Webhook integrieren, keine Secrets im Frontend.

Das bestehende Branding, der Titel und die Grundgestaltung sollen erhalten bleiben.
