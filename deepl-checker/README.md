# DeepL Glossar Checker

Kleines Testprojekt zum Prüfen, ob ein DeepL-Glossar korrekt verwendet wird.

## Lokal starten

```bash
npm install
npm start
```

Danach `http://localhost:3000` öffnen.

Der API-Key wird nicht gespeichert. Er wird nur vom Browser an diesen Testserver und von dort an DeepL weitergegeben.
